 TeamWave 
 es una plataforma para CRM, gestión de recursos humanos y gestión de proyectos. Beetrack proporciona seguimiento de entregas en tiempo real para envíos. La API de Beetrack Rastrear esta API permite a los desarrolladores administrar datos de entrega y logística, incluidas rutas, despacho, vehículo, GPS, datos, contactos, grupos y elementos de guía. Beetrack tiene su sede en Santiago, Chile. La API aparece en Envío .

Applitools 

habilita herramientas para crear y monitorear aplicaciones móviles, web y nativas. Applitools es un motor de nube basado en inteligencia artificial que valida los aspectos visuales de cualquier aplicación de forma totalmente automatizada. Esta plataforma garantiza que las pantallas no cambien inesperadamente mediante el uso de pruebas de regresión. La API REST de Applitools Track esta API integra flujos de pruebas visuales con aplicaciones. Está archivado en nuestra categoría DevOps .
